## 数据流

```
前端（PublishCenter.vue）
  └─ 正文输入框 ──→ tab.description
  └─ publishData.description ──→ POST /postVideo

后端（sau_backend.py）
  └─ /postVideo ──→ data.get('description', '')
  └─ post_video_DouYin(..., description)

后端（myUtils/postVideo.py）
  └─ post_video_DouYin(..., description='')
  └─ DouYinVideo(..., desc=description)

抖音上传器（uploader/douyin_uploader/main.py）
  └─ fill_title_and_description()
     ├─ 填入标题（title_input.fill(title[:30])）
     ├─ 填入正文（page.keyboard.type(description)）
     └─ 追加话题（page.keyboard.type(" #tag") → 已有逻辑）
```
